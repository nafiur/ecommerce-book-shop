# রোকমারি মোবাইল মেনু ইন্সট্রাকশন

## মোবাইল মেনু যুক্ত করার ধাপ:

### 1. HTML ফাইলে মোবাইল মেনু যুক্ত করুন

`mobile-menu-snippet.html` ফাইলের কন্টেন্ট কপি করে আপনার HTML ফাইলের `<body>` ট্যাগের ঠিক পরে paste করুন। উদাহরণ:

```html
<body>
    <!-- Mobile Menu Overlay -->
    <div class="mobile-menu-overlay"></div>

    <!-- Mobile Offcanvas Menu -->
    <div class="mobile-menu">
        ...মেনুর বাকি অংশ...
    </div>

    <!-- বাকি পেজের কন্টেন্ট -->
    ...
</body>
```

### 2. মোবাইল মেনু টগল বাটন যুক্ত করুন

হেডার সেকশনে, আইকন এরিয়াতে একটি হ্যামবার্গার মেনু বাটন যুক্ত করুন (এটি শুধুমাত্র মোবাইলে দৃশ্যমান হবে):

```html
<!-- Mobile Menu Toggle (visible only on mobile) -->
<button class="btn btn-link text-dark d-lg-none me-2" id="mobileMenuToggle">
    <i class="fas fa-bars fs-4"></i>
</button>
```

### 3. সম্পন্ন!

এখন CSS এবং JavaScript ইতিমধ্যে যুক্ত হয়ে গেছে। আপনার মোবাইল মেনু কাজ করবে।

## ফাইলগুলো যেখানে পরিবর্তন করা হয়েছে:

1. ✅ `assets/css/style.css` - মোবাইল মেনুর স্টাইল যুক্ত
2. ✅ `assets/js/main.js` - মেনু টগল ফাংশনালিটি যুক্ত
3. ⏳ `index.html` - মোবাইল মেনু HTML যুক্ত করতে হবে
4. ⏳ অন্যান্য HTML ফাইল - প্রয়োজন অনুযায়ী
